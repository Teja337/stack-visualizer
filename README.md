# Stack Visualizer 🚀

Interactive stack visualization tool supporting:

✔ Array Stack
✔ Linked List Stack
✔ Push / Pop / Peek / Clear
✔ Animated UI
✔ Dark/Light Theme
✔ Top Pointer Visualization

## Tech Used
- HTML
- CSS
- JavaScript
- Backend API

## Features
- Real-time stack visualization
- Error handling
- Stack overflow handling
- Modern UI
